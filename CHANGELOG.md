# Changelog

All notable changes to Pubcraft will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] — 2026-04-28

### Added
- **Part 8 restructured into platform-specific subsections** with a universal "social rules" intro (Part 8.0).
- **X (Twitter) coverage (Part 8.2):** open-source Grok ranking weights (reply ≈ 27x like, conversation ≈ 150x like), Premium boost economics (2x out-of-network, 4x in-network), long-form vs. thread tradeoffs, sentiment penalties, hook patterns.
- **Reddit coverage (Part 8.3):** subreddit-fit decision tree, specificity test for titles, body-craft conventions (first-person, no CTA, disclosure rules), the comment-led marketing strategy, the branded-subreddit play (Eric Eden pattern), banned moves that get accounts shadowbanned.
- **Product Hunt coverage (Part 8.4):** post-2024 curated featuring reality (10% featuring rate, judged on usefulness/novelty/craft/creativity), the full five-piece launch copy package (tagline / description / gallery / first comment / response templates), tagging strategy, timing.
- **Cross-platform repurposing rules (Part 8.5).**
- Updated YAML description and "When to use" section to trigger on X, Reddit, and Product Hunt requests.
- Updated Part 9 with platform-specific routing instructions.

### Changed
- LinkedIn section is now Part 8.1 (was the entirety of Part 8).
- Part 9 now routes to the right platform-specific subsection based on requested output type.
